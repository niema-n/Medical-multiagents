# Base documentaire médicale locale

Ajoute ici des fichiers `.md` médicaux validés pour ton projet PFA.

Exemples de thèmes utiles :

- symptômes respiratoires
- douleur thoracique
- fièvre
- signes de gravité
- recommandations intermédiaires prudentes

Ensuite, indexe les documents avec :

```powershell
python -m backend.app.rag.index_documents
```
